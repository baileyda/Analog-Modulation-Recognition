This is a system for recognition of analog modulation scheme, it is based on "Automatic Modulation Recognition of Communication Signals" by Azzouz and Nandi

Make use of some .mp3 named "02.mp3" and http://www.csun.edu/~skatz/katzpage/sdr_project/sdr/am_usrp710.dat.zip recorded by Sharlene Katz from the California State University
